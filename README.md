# Food Construct — лабораторная работа 3

В проекте реализованы:

- главная страница `index.html`;
- страница «Собрать ланч» `lunch.html`;
- карточки блюд с адаптивной сеткой;
- форма заказа после блока с меню;
- отправка формы методом `POST` на `https://httpbin.org/post`;
- обязательные поля, селекты, textarea, checkbox, radio, time, reset и submit;
- общие стили `style.css`;
- стили страницы «Собрать ланч» и формы `lunch.css`;
- адаптивность для планшетов и мобильных устройств.

Для публикации через GitHub Pages файл `index.html` должен лежать в корне репозитория.
