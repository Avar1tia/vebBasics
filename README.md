# Food Construct — лабораторная работа 4

Сайт компании по доставке бизнес-ланчей.

## Страницы

- `index.html` — главная страница.
- `lunch.html` — страница «Собрать ланч» с динамическим выводом блюд и формой заказа.

## JavaScript

- `dishes.js` — массив объектов с блюдами.
- `script.js` — вывод карточек на страницу, сортировка блюд, выбор блюд, отображение блока «Ваш заказ» и подсчет стоимости.

## Особенности

- Карточки блюд создаются динамически через JavaScript.
- В HTML изначально нет карточек блюд.
- У каждой карточки есть `data-dish`.
- Блюда сортируются по алфавиту внутри категорий.
- При выборе блюда оно появляется в блоке «Ваш заказ».
- При выборе другого блюда той же категории оно заменяет предыдущее.
- Если выбрана только часть блюд, для остальных категорий выводится сообщение, что блюдо не выбрано.
- Если ничего не выбрано, отображается «Ничего не выбрано».
- Считается итоговая стоимость заказа.
- Форма отправляется методом POST на `https://httpbin.org/post`.
