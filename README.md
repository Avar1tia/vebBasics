# Food Construct — лабораторная работа 5

В проекте выполнены страницы сайта компании по доставке бизнес-ланчей.

## Страницы

- `index.html` — главная страница.
- `lunch.html` — страница «Собрать ланч».

## Что реализовано

- Блюда хранятся в отдельном файле `dishes.js`.
- Карточки блюд создаются динамически с помощью JavaScript.
- В каждой категории по 6 блюд.
- Добавлены категории «Салаты и стартеры» и «Десерты».
- Для каждого блюда добавлено свойство `kind`.
- Для каждой категории добавлены фильтры с атрибутом `data-kind`.
- Фильтры работают независимо для каждой категории.
- При повторном клике по активному фильтру он отключается.
- Блюда сортируются по алфавиту.
- Выбранные блюда добавляются в раздел «Ваш заказ».
- Итоговая стоимость заказа пересчитывается автоматически.
- Форма отправляется методом POST на `https://httpbin.org/post`.
- Сохранена адаптивность страницы.

## Файлы стилей

- `style.css` — общие стили сайта.
- `lunch.css` — стили страницы «Собрать ланч», карточек, фильтров и формы заказа.
