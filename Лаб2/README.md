# Food Construct — лабораторная работа 2

В архиве находятся две страницы сайта:

- `index.html` — главная страница;
- `lunch.html` — страница «Собрать ланч»;
- `style.css` — общие стили;
- `lunch.css` — дополнительные стили страницы «Собрать ланч»;
- `images/` — изображения блюд.

## Как открыть

Откройте `index.html` в браузере. Через меню можно перейти на страницу «Собрать ланч».
